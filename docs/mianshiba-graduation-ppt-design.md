# 面试吧项目答辩 PPT 设计稿

## 基本定位

- 答辩时长：10-15 分钟。
- PPT 作用：负责引导讲解，具体完成效果切到前端展示。
- 页数建议：15 页。
- 叙事主线：技术求职准备碎片化 → 面试吧提供 AI 求职闭环 → 前端展示完成效果。

## 统一视觉风格

- 风格：和项目前端保持一致，使用 Neubrutalism 风格。
- 背景：暖白色 `#FEF9EF`。
- 主色：紫色 `#6C5CE7`。
- 装饰：黑色粗边框、硬阴影、卡片式布局。
- 每页右下角放“前端演示路径”，方便上台切换。

## 演示路线

```text
/login
→ /
→ /resume
→ /resume/:id/edit
→ /interview/new
→ /interview/:id/room
→ /interview/:id/report
→ /training
→ /applications
→ /coach
→ /analytics
→ /admin
```

### 第 1 页：封面

标题：面试吧 AI 面试模拟平台

副标题：面向技术求职者的 AI 求职训练与投递管理系统

页面内容：

- 毕业设计答辩
- 姓名、专业、指导老师、日期
- 技术栈标签：Vue 3、Spring Boot、MySQL、Redis、DeepSeek

- 居中大标题。

### 第 2 页：项目背景

标题：技术求职准备流程碎片化

- 八股训练缺少针对性。
